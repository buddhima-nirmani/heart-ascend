Heart Ascend - A Distributed Quiz Challenge

🎮 Overview:

Heart Ascend is a high-energy web-based quiz game where players test their speed and accuracy by counting hearts against a ticking clock. Built as part of a Distributed Service Architecture module, the game leverages cloud technology to provide a seamless, real-time competitive experience.

🚀 Features:

Real-time Leaderboard: Global ranking system powered by Firebase Firestore.
Secure Authentication: User signup and login systems to track individual progress.
Dynamic Difficulty: Custom ranking system ranging from "Heart Novice" to "Cardio Master."
Responsive Design: Glassmorphism UI with animated backgrounds for an immersive feel.

🛠️ Tech Stack:

Frontend: HTML5, CSS3 (Custom Glassmorphism), JavaScript (ES6)
Backend as a Service (BaaS): Firebase (Authentication & Cloud Firestore)
APIs: Heart Game API integration for dynamic content.

📋 Project Management Insights:

Database Schema: Designed a flat NoSQL structure for the players collection to optimize read/write speeds for the leaderboard.
Security: Implemented server-side security rules to ensure data integrity and user isolation.
Architecture: Followed a modular JavaScript approach to separate game logic from audio and session management.
